# Development Authentication Guide

> Generated by muscomp — DO NOT EDIT

## Quick Start

For local development, compile with `-DMUSCOVITE_DEV_AUTH=ON` to enable
the `DevAuthProvider` which accepts a static bearer token.

### Test credentials

| User | Token | Roles |
|------|-------|-------|
| dev-admin | `muscodev` | All roles (owner + app + readonly) |
| dev-readonly | (programmatic) | Readonly on all schemas |

### Usage with grpcurl

```bash
# List services
grpcurl -plaintext localhost:50051 list

# Call with dev token
grpcurl -plaintext \
  -H "Authorization: Bearer muscodev" \
  localhost:50051 muscovite_harbor.version.VersionService/GetVersion
```
